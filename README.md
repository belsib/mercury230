# ESPHome компонент для подключения счетчиков электроэнергии Меркурий 230, 206

* [1. Назначение](#1-назначение)
* [1. Тестирование](#1-тестирование)

## 1. Назначение
Данный компонент предназначен для считывания параметров электроэнергии с электросчетчиков Меркурий, поддерживающих проприеритарные протоколы Меркурий 1 и Меркурий 3.

## 1. Тестирование
Компонент протестирован на электросчетчиках Меркурий 206, 230.
